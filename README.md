# Mary-Wollstonecraft-Shelley---Landing-Page

## Sobre o projeto

Este projeto consiste na criação de uma **landing page** dedicada mulheres que mudaram o mundo, a mulher que a estudante escolheu foi a escritora **Mary Shelley**, autora da obra *Frankenstein*.

A página foi desenvolvida utilizando tecnologias básicas da web: **HTML, CSS e JavaScript**, com o objetivo de apresentar informações sobre a autora de forma organizada e interativa.

---

## Objetivo

O principal objetivo do projeto é:

* Aplicar conhecimentos de **HTML** para estruturar o conteúdo.
* Utilizar **CSS** para estilização da página.
* Implementar **JavaScript** para adicionar interatividade.
* Desenvolver habilidades de pesquisa e organização de informações.

---

## Estrutura da página

A landing page contém as seguintes seções:

* **Biografia**
  Breve resumo da vida de Mary Shelley.

* **Principais Realizações**
  Destaque de suas contribuições, como a criação de *Frankenstein*.

* **Curiosidades (Interativo)**
  Botão que exibe curiosidades aleatórias sobre a autora.

* **Desafios**
  Dificuldades enfrentadas ao longo de sua vida.

* **Datas Importantes**
  Eventos marcantes de sua trajetória.

* **Obras**
    Algumas das obras mais importante da autora.

* **Imagens**
  Galeria com imagens relacionadas à autora e sua obra.

---

## Funcionalidade em JavaScript

A principal interação da página é o botão de curiosidades:

* Ao clicar no botão, uma curiosidade aleatória é exibida
* A cada novo clique, uma nova curiosidade aparece
* A interação é simples e dinâmica, tornando a navegação mais interessante

---

## Tecnologias utilizadas

* **HTML5** → estrutura da página
* **CSS3** → estilização
* **JavaScript** → interatividade

---

## Considerações finais

Este projeto foi desenvolvido com foco no aprendizado das bases do desenvolvimento web.

Além de praticar programação, também permitiu conhecer melhor a importância histórica e literária de Mary Shelley.
